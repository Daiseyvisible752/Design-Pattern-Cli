## 🚀 What this tool does

Design-Pattern-Cli is a command-line tool that helps you create starter code for common GoF design patterns.

It can generate boilerplate for patterns such as:
- Singleton
- Factory Method
- Abstract Factory
- Builder
- Adapter
- Strategy
- Observer
- Decorator
- Facade
- Command

You can use it to save time when you need a clean starting point for a project in languages like:
- Go
- Python
- C#
- Java

The tool is useful when you want:
- a fast project setup
- a consistent folder layout
- simple code templates for design patterns
- less repetitive setup work

## 🛠️ Common patterns supported

### Singleton
Creates a single shared instance setup.

### Factory Method
Creates starter code for building objects through a common method.

### Abstract Factory
Creates a family of related objects.

### Builder
Creates code for step-by-step object setup.

### Adapter
Creates code that lets two different interfaces work together.

### Strategy
Creates code for swapping behavior at run time.

### Observer
Creates code for one object that notifies others when something changes.

### Decorator
Creates code for adding behavior without changing the main object.

### Facade
Creates code for a simple front layer over a larger system.

### Command
Creates code for wrapping actions into objects.
